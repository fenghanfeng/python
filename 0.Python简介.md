### 2.基础语法

```
1： 缩进
    Python中，四个字符为一个缩进，Python中没有{ }, 使用缩进替代
2：续行
    Python中，在行的末尾使用\
3：字符转义
    \\， \n, \t:    \\将\当做普通字符对待
4：标识符
    只能以字母或下划线开头
    只能是字母、下划线、数字
    不能使用Python中的关键字
    区分大小写
5：常量
    Python中没有常量，通常使用大写字母表示常量
6：变量
    赋值后可以改变其值的标识符
```
